# GPUを用いた偏微分方程式の差分法数値シミュレーション

## 概要

- 差分法で偏微分方程式の数値シミュレーションをします。
- 差分法の拡散方程式の漸化式は、以下のようになります。
- u\[i\]\[j\] = (1 - 4 * r) * u\[i\]\[j\] + r * (u\[i + 1\]\[j\] + u\[i - 1\]\[j\] + u\[i\]\[j + 1\] + u\[i\]\[j - 1\]);
